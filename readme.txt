1️⃣ What is the difference between var, let, and const?

Var was used in old days. const is used when we know the variable will not be changed. Let is used when the variable might get changed.


2️⃣ What is the spread operator (...)?

Spread operator separates each item from an array/object as individual 


3️⃣ What is the difference between map(), filter(), and forEach()?

map() : creates new array by applying function
forEach() : it executes function for every index of the array
filter() : creates new array containing the elements which passes the certain condition 


4️⃣ What is an arrow function?

Arrow functions provide a shorter syntax for writing functions. 


5️⃣ What are template literals?

Template literals use `` instead of "". Helps to write multi-line. No hassle of concatenation.
